Parameters
detection_model_path- Path to a binary .pb file containing trained detector network.
confidence_threshold- Confidence threshold for considering text detected in an image region.
nms_threshold- Non-maximum suppression threshold for detection.
recognition_model_path- Path to a binary .onnx file containing trained CRNN text recognition model.
vocabulary_path- Path to vocabulary benchmarks for evaluation.
resize_width- Preprocess input image by resizing to a specific width. It should be a multiple of 32.
resize_height- Preprocess input image by resizing to a specific height. It should be a multiple of 32.
